# Code for Rohden et al. (2025): "Glial reactivity is linked to synaptic dysfunction across the aging and Alzheimer’s disease continuum"
R code of the data analysis for the paper *"Glial reactivity is linked to synaptic dysfunction across the aging and Alzheimer’s disease continuum".*

## Required R packages

```r
library(arm)
library(MASS)
library(merTools)
library(lme4)
library(broom)
library(ggplot2)
library(tidyr)
library(sjPlot)
library(sjmisc)
library(dplyr)
library(lmerTest)
library(wesanderson)
library(effects)
library(MuMIn)
library(lsmeans)
library(glmmTMB)
library(optimx)
library(readxl)
library(dplyr)
library(table1)
```
